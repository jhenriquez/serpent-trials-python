# Trial 11 - Comida y colisiones con Turtle

## 1. Objetivo

Agregar comida y detectar colisiones en la version grafica.

## 2. Explicacion del concepto

La logica es parecida a la del juego de terminal:

- la comida aparece en una posicion
- la serpiente puede alcanzarla
- el juego termina si hay colision

La diferencia es que ahora lo veras todo en una ventana.

## 3. Conexion con Snake

Este trial lleva las reglas del juego completo a la version de Turtle.

Con eso, la version grafica deja de ser solo una demostracion visual y se convierte en un juego con objetivo y riesgo.

## 4. Requisito concreto

Abre `projects/snake-turtle/main.py` y toma tu version del trial anterior.

Luego:

1. Agrega una comida visible.
2. Detecta cuando la cabeza toca la comida.
3. Haz crecer la serpiente al comer.
4. Detecta colision con los bordes.
5. Detecta colision con el cuerpo.
6. Termina la partida cuando haya choque.

Ejecuta:

```bash
python projects/snake-turtle/main.py
```

## 5. Resultado visible esperado

Debes ver comida en la ventana y un final de partida cuando la serpiente choca.

## 6. Desafio opcional

Haz que la comida aparezca en posiciones nuevas sin tocar el cuerpo de la serpiente.
