# Trial 10 - Cuerpo de la serpiente con Turtle

## 1. Objetivo

Representar varias partes de la serpiente en la ventana grafica y hacer que sigan a la cabeza.

## 2. Explicacion del concepto

Igual que en la version de terminal, la serpiente necesita varias posiciones.

La diferencia es que ahora cada parte puede ser una figura grafica.

Puedes guardar esas partes en una lista y actualizar sus posiciones cuando la cabeza se mueve.

## 3. Conexion con Snake

La cabeza sola no basta.

Para que se sienta como Snake, el cuerpo debe seguirla.

Este trial reconstruye esa idea en la version visual.

## 4. Requisito concreto

Abre `projects/snake-turtle/main.py` y toma tu version del trial anterior.

Luego:

1. Crea una lista para guardar los segmentos del cuerpo.
2. Agrega uno o mas segmentos visibles.
3. Mueve la cabeza.
4. Haz que cada segmento siga a la parte que va delante.

La meta es que se note claramente una serpiente con varias partes.

Ejecuta:

```bash
python projects/snake-turtle/main.py
```

## 5. Resultado visible esperado

Debes ver una cabeza y uno o mas segmentos que se mueven detras de ella.

## 6. Desafio opcional

Haz que la cabeza y el cuerpo tengan colores distintos.
