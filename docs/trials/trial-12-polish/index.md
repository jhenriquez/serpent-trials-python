# Trial 12 - Pulido final

## 1. Objetivo

Cerrar la version grafica con puntaje, mensaje de fin de juego y opcion de reinicio.

## 2. Explicacion del concepto

Pulir un juego significa mejorar la experiencia final:

- mostrar informacion clara
- cerrar una partida de forma entendible
- permitir volver a jugar

No es una mecanica nueva, pero si hace que el proyecto se sienta completo.

## 3. Conexion con Snake

Ya tienes la base del juego grafico.

Ahora vas a dejarlo listo como una experiencia final que el jugador puede entender y repetir.

## 4. Requisito concreto

Abre `projects/snake-turtle/main.py` y toma tu version del trial anterior.

Luego:

1. Muestra el puntaje en la ventana.
2. Muestra un mensaje claro de fin de juego.
3. Permite reiniciar sin cerrar el programa.
4. Vuelve a crear el estado inicial cuando el jugador quiera jugar otra vez.

La meta es terminar una version grafica completa y jugable.

Ejecuta:

```bash
python projects/snake-turtle/main.py
```

## 5. Resultado visible esperado

Debes ver un Snake grafico completo con puntaje, fin de juego y reinicio.

## 6. Desafio opcional

Agrega una pantalla inicial simple con un mensaje como `Press any key to start`.
