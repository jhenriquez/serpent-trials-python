# Product Requirements Document

## Product

Serpent Trials is a progressive Python course for young beginners who learn core programming concepts by building a Snake game.

## Audience

- Around 13 years old
- Beginner level
- Comfortable using a computer, folders, and a basic terminal

## Primary Deliverables

- A Spanish-language course in markdown
- A terminal ASCII Snake project built progressively
- A turtle graphics Snake project built after the terminal version

## Functional Requirements

- Each trial must produce visible progress
- Each trial must keep the project runnable
- The terminal version must be completed before the turtle version begins
- Final terminal and turtle projects must run independently

## Non-Goals

- No external dependencies beyond `turtle`
- No advanced architecture patterns
- No performance optimization work
- No browser-based gameplay

## Success Criteria

- Students can complete the course with minimal friction
- Students see progress after every trial
- Both final Snake projects are playable
