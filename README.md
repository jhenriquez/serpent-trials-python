# Serpent Trials

Serpent Trials is a beginner-friendly Python course where students build Snake step by step.

The repository is organized around:

- `docs/` for GitHub Pages course navigation
- `trials/` for Spanish lesson content
- `projects/` for the runnable game code
- `scaffolding/` for starter templates

Current status:

- Initial repository scaffold created
- Terminal project has a runnable placeholder
- Turtle project has a runnable placeholder
- Trial 0 is drafted

## Run the starter project

```bash
python projects/snake-terminal/main.py
```

## Course path

1. Trial 0: setup and first run
2. Trials 1-8: terminal Snake
3. Trials 9-12: turtle Snake
